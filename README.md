# 保溫杯探究互動教程

這是一份國小自然科學互動教材，依據「製作保溫杯」教學影片與逐字稿整理而成。教材引導學生比較不同包覆材料的保溫效果，並設計更薄、更方便、效果更好的保溫裝置。

## 教材結構

- `觀察站`：學習目標、實驗器材、重要詞彙。
- `實驗示範`：熱水降溫與冰水升溫兩組資料比較。
- `原理小教室`：用 Canvas 動畫說明傳導、對流、輻射與材料隔熱。
- `生活應用`：保溫杯、便當袋、建築隔熱、救生毯、冰桶、外送保溫箱。

## 主要資料

- 熱水實驗：80 度熱水靜置 20 分鐘。
- 冰水實驗：6.2 度冰水靜置 30 分鐘。
- 結論：棉布組的溫度差最小，保溫效果最佳。
- 設計提醒：當保溫效果相近時，厚度越薄的保溫裝置較佳。

## 使用方式

直接開啟 `index.html` 即可瀏覽；若要用本機伺服器預覽：

```powershell
python -m http.server 7788
```

然後開啟 `http://localhost:7788/`。
